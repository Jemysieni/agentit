Palautteenantoagentti
Tämä sivu on rakennettu Microsoft 365 Agents SDK -integraatiota varten. Kun kirjautuminen ja agenttiyhteys on otettu käyttöön, keskustelukenttä ilmestyy tähän.
Valmis pohja. Seuraavaksi tarvitaan Agents SDK -JavaScript, Entra ID -kirjautuminen ja Copilot Studio -connection string.

Keskustelukenttä tulee tähän

Tämä sivu toimii käyttöliittymäpohjana Copilot Studio -agentille. Native app / Microsoft 365 Agents SDK -mallissa varsinainen agenttiyhteys tehdään JavaScript-sovelluksessa, joka:

  1. kirjaa käyttäjän sisään Microsoft Entra ID:llä
  2. hakee oikeudet Copilot Studio -agentin kutsumiseen
  3. muodostaa yhteyden connection stringin avulla
  4. renderöi agentin WebChatin tähän näkymään

Kun SDK-logiikka on lisätty, tämä placeholder korvautuu oikealla keskustelukentällä.

Muistilista seuraavaa vaihetta varten
  • Copilot Studio → Channels → Native app → Microsoft 365 Agents SDK → connection string
  • Microsoft Entra ID app registration
  • API-oikeus: CopilotStudio.Copilots.Invoke
  • JS-sovellus, joka käyttää Agents SDK:ta + WebChatia
